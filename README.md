# doulos-han-dialect

An extended version of Doulos-based font with PUA glyphs for Sinitic dialectological transcription.

基于Doulos字体的扩展版字形，包含用于汉语方言转写的私用区位符号。

## Overview

This project provides a modified version of a Doulos-based font, extended with additional glyphs encoded in the Unicode Private Use Area (PUA, U+E000–).

The added glyphs are designed primarily for Sinitic dialectological research, including phonetic transcription and fieldwork notation that are not covered by standard Unicode.

本项目提供基于Doulos字体的改良版本，扩展了额外字形并将其编码于Unicode私有使用区（PUA，U+E000–）。新增字形主要服务于汉语方言学研究，涵盖标准Unicode未收录的音标转写与田野调查标记体系。
